# 🛠️ Helpdesk System

Sistema de chamados (Helpdesk) desenvolvido em **C#**, com aplicação de conceitos de **programação orientada a objetos (POO)** e menus interativos via console.

Este projeto faz parte de um conjunto de aplicações desenvolvidas com foco em **aprendizado prático** e **portfólio profissional**.

---

## 💡 Funcionalidades

- 📌 Abertura de chamados
- 📋 Listagem de chamados
- ✅ Encerramento de chamados
- 🧑‍💻 Interface via console
- 🔄 Organização em classes e métodos

---

## 🚀 Como executar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/Willm2709/helpdesk-system.git
   ```

2. Abra o projeto no **Visual Studio** (arquivo `.sln`)

3. Compile o projeto:
   - `Ctrl + Shift + B` (ou menu **Compilar > Compilar Solução**)

4. Execute:
   - `Ctrl + F5` para rodar sem depuração
   - `F5` para rodar com depuração
